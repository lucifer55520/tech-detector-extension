# Tech Detector Extension

A browser extension that detects technologies used on websites such as frameworks, CMS, and libraries.

## Features
- Detects frontend and backend technologies
- Simple UI
- Helpful for developers and cybersecurity learners

## Installation
1. Download this repository
2. Open Firefox
3. Go to Extensions → Debug Add-ons
4. Click “Load Temporary Add-on”
5. Select manifest.json

## Project Link
https://addons.mozilla.org/en-US/firefox/addon/tech-detector/
